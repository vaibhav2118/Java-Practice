## 🛠️ Getting Started

To run the programs in this repository, you'll need the **Java Development Kit (JDK)** installed on your machine.

### How to Run

1. **Compile the file:**
   ```powershell
   javac FileName.java
   ```

2. **Run the program:**
   ```powershell
   java FileName
   ```
